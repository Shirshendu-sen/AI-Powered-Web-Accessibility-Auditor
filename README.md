# AI-Powered Web Accessibility Auditor

Scans a public URL, reports WCAG violations, and proposes concrete before/after
fixes (deterministic contrast fixes, AI-generated alt-text and ARIA fixes).

Canonical documentation lives in [`docs/guide.md`](docs/guide.md) — the phase-by-phase
execution guide is the single source of truth for architecture, scope, credentials,
and build order. Start there.
